## 📖 使用指南

### 1. 文献查询

**按类别查询**:

2. 浏览该类别下的所有文献

**按概念查询**:

2. 查找相关概念
3. 根据概念定位相关文献

### 2. 知识图谱使用

**JSON格式**:
- 可直接导入到图数据库（如Neo4j）
- 可用于可视化工具（如D3.js、Cytoscape）
- 可用于AI知识推理

**示例代码** (Python):
```python
import json

# 读取知识图谱
with open('tcm_knowledge_graph.json', 'r', encoding='utf-8') as f:
    kg = json.load(f)

# 查询某概念的所有关系
concept_name = "心"
relations = [edge for edge in kg['edges']
             if edge['from_name'] == concept_name or edge['to_name'] == concept_name]

for rel in relations:
    print(f"{rel['from_name']} --[{rel['type']}]--> {rel['to_name']}")
```

### 3. 公理推理

**推理流程**:
```
症状输入
    ↓
八纲辨证 (R1) → 表里、寒热、虚实、阴阳
    ↓
脏腑辨证 (R2) → 病位确定
    ↓
病机分析 → 应用公理A1-A5
    ↓
治则确定 (D3) → 正治/反治
    ↓
方药选择
```
